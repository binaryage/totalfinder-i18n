Sie können Ihre Lizenz auf zwei unterschiedlichen Arten verwenden:

1. Ein Benutzer auf mehreren Computern (solange Sie die einzige Person sind, die TotalFinder verwendet)
Beispiel: Ein Desktop-Rechner und ein persönliches Laptop, die beide ausschließlich von Ihnen verwendet werden
2. Mehrere Benutzer auf einem einzigen Computer
Beispiel: Ein Rechner zu Hause, der von allen Familienmitgliedern genutzt wird

Weitere Informationen zur Lizensierung finden Sie unter http://totalfinder.binaryage.com/licensing
